# 🛒 Store Sales Forecasting Project

## 📌 Overview
This project predicts retail store sales using historical data from a Kaggle dataset.  
The goal is to analyze trends and build a machine learning model for forecasting.

---

## ⚙️ Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

## 📊 Dataset
Kaggle Store Sales Forecasting Dataset  
https://www.kaggle.com/competitions/store-sales-time-series-forecasting

---

## 🔍 Project Steps
1. Data Loading  
2. Data Cleaning  
3. Exploratory Data Analysis (EDA)  
4. Feature Engineering  
5. Model Building  
6. Evaluation  

---

## 📈 Results
- Built a predictive model for sales forecasting  
- Identified trends and seasonal patterns
